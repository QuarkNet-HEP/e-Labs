Here's a simple Plotly plot.

You can use this page as a playground by opening the console and typing:

// clear out the existing data so Plotly.plot will accept a new layout
delete TESTER.data;
delete TESTER.layout;
// try out any data and layout objects - see tests.html for examples
Plotly.plot(TESTER,[
    {x:[1,3,5,7,9],y:[2,5,7,3,8],type:'bar'},
    {x:[1,3,5,7],y:[4,6,3,1],type:'bar'}
    ],{plot_bgcolor:'#EEE'});
// check out the data and layout objects
TESTER.data
TESTER.layout
// modify them
Plotly.relayout(TESTER,{'xaxis.range[1]':15, 'barmode':'group'})
Plotly.restyle(TESTER,{bardir:'h'});
