Milestone: Use search parameters to find data of interest.

To pursue your research question, you must find data that "fits" your study by searching the cosmic ray database for relevant data.

Setting parameter values allows you to select data files for analysis. These parameters can include:

  • Date - looking for coincident times.
  • Location - looking for near-by detectors: state? city? school?
  • Channels - that is, 1, 2, 3, 4.
  • Detector configuration - stacked? unstacked?
