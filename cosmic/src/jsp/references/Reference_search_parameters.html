Milestone: Know how to use search parameters to find data of interest.

To pursue your research, you must search the cosmic ray database for relevant data that "fits" your study.

You set the following parameters when you choose which date files to analyze.

  • Date - looking for coincident times.
  • Location - looking for near-by detectors: state? city? school?
  • Detector configuration - stacked? unstacked?

You set other parameters after you click the "Run Analysis" button.
contains additional information

You will probably run your analysis several times for each data set, changing the parameters each time.

Keep track of the parameters you select in your logbook.
