Milestone: Know how to use your analysis tools and how to search for data of interest.

The e-Lab provides online analysis tools. To pursue your research, you must search the cosmic ray database for relevant data that "fits" your study. These analyses are most likely more complex than any analyses you have done before.

Links to tutorials and step-by-step instructions are located on study page.
contains additional information about the parameters. You set the following parameters when you choose which date files to analyze.

  • Date - Looking for coincident times.
  • Location - Looking for nearby detectors: state? city? school?
  • Detector configuration - Stacked? unstacked?

You set the following parameters after you click the "Run Analysis" button.

  • Analysis controls
  • Plot contorls
  • Fit controls (Lifetime only)
  • Execution mode, where you want to run the analysis

You will probably run your analysis several times for each data set, changing the parameters each time.

Use the navigation bar to find your study.

Keep track of your different analyses in your logbook.
