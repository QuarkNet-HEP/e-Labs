Milestone: Choose and use appropriate analysis tools.

A clear research question couples your selected data with analysis tools. There are four basic tools; which one you use depends on the information you are seeking. You will execute these analysis routines on the Grid.

The analysis tool bag contains:

  • Performance Study - detector performance check.
  • Lifetime Study - the particles in our detector.
  • Flux Study - rate that cosmic rays hit a given area over time.
  • Shower Study - multiple detectors lighting up at the same time.
