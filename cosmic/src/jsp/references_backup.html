Reference_analysis_tools A clear research question couples your selected data with analysis tools. There are four basic tools; which one you use depends on the information you are seeking. You will execute these analysis routines on the Grid.

The analysis tool bag contains:

  • Performance Study - detector performance check.
  • Lifetime Study - the particles in our detector.
  • Flux Study - rate that cosmic rays hit a given area over time.
  • Shower Study - multiple detectors lighting up at the same time.

Reference_search_parameters To pursue your research question, you must find data that "fits" your study by searching the cosmic ray database for relevant data.

Setting parameter values allows you to select data files for analysis. These parameters can include:

  • Date - looking for coincident times.
  • Location - looking for near-by detectors: state? city? school?
  • Channels - that is, 1, 2, 3, 4.
  • Detector configuration - stacked? unstacked?
