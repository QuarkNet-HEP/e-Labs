Step-by-Step Instructions: Performance Study

Data Button - Performance Button - Searching Data Files - Selecting Data - Analyzing
The Plot - View Plot Controls - Set Controls - Saving a Plot - Finishing

  • Keep two windows open, one with these instructions and the other with the Data web pages.
  • Go back and forth between them.
  • Use the scroll bar to move the instructions.
  • The Performance Study allows you to study the quality of the data.
  • Look for the blue highlights for guidance.

We shall create a performance plot. The data file is from Spearfish High School, dated Jan. 8, 2015.

Step 1:Click on the Data button in the main menu.
Step 2:Click on the Performance button in the submenu. You will see a search form.
Step 3.1:Look at the search page. On the right, we have a button to run the study and links to a tutorial, step-by-step instructions, FAQs and related milestones. The legend shows icons for information about each data file.
Step 3.2:Search for a data file from Spearfish High School with starting date 01/01/2015.
a) Select School in the pulldown list.
b) Enter Spearfish High School in the search field.
c) Click on the little arrow next to "Advanced Search".
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "01/01/2015" in the first data field.
b) Click Search Data.
Step 3.4:a) Click on the little arrow next to "Spearfish High School" to see the data files.
Step 3.5:a) Click on the little arrow next to "January 2015" to see the data files for that month.
Step 3.6: All the data files uploaded for January are displayed. See the data by clicking on the magnifying glass icon next to the date and additional information by clicking on the comments bubble.
Step 4: a) To select data for the analysis, select the check box next to "Thu 08."
b) Click the Run performance study button on the right.
Step 5:Look at the number of events in each channel; they are all about the same numbers. This shows that the detectors are well-plateaued.
a) Look at the Analysis Controls below. Click on the Question Mark next to Bin width for an explanation of it and why 2.5 is a good choice for analyzing this data that comes from a Series 6000 DAQ. Change the Bin width (ns) to 2.5.
b) Click Analyze.
Step 5.1:If the analysis takes very little time to run, the plot may show up quickly. More often, though, one gets a progress bar displaying how the analysis is doing. One can click on Queue study to allow the analysis to continue in the background.
Step 5.2:Click on Analyses on the navigation bar to see the job's progress.
Step 5.3:When the job is done, an entry will appear in the Analysis List like this. Click on the link and see the plot.
Step 6:Here's the plot of the performance (a histogram showing the frequency of the signal widths on 01/08/2015). How will this plot looks if we make it smaller?
Click Change.
Step 7:Click on the small arrow to view the Plot Controls.
Step 8:a) Select Small for the size of the plot.
b) Click Analyze.
Step 9:Enter a name for the plot similar to, but not the same as, the one shown. Click Save Plot. Each time through this "step-by-step", choose a different name for the plot.
This plot can be accessed later for the poster.
Step 10:Note the successfully saved plot. Click Close.
One can create more plots for other channels or go back to choosing a new data set by clicking Performance. Click View Plots at any time to see the plots.
