Step-by-Step Instructions: Performance Study

Data Button - Performance Button - Searching Data Files - Selecting Data - Analyzing
The Plot - View Plot Controls - Set Controls - Saving a Plot - Finishing

Keep two windows open, one with these instructions and the other with the Data web pages. Go back and forth between them. Use the scroll bar to move the instructions. The Performance Study allows you to study the quality of the data. Look for the orange highlights for guidance.

Our goal here is to create a performance plot. The data for this plot is from Seaman High School and dated 2/05/09.

Step 1:Click on the Data button in the main menu.
Step 2:Click on the Performance button in the submenu. You will see a search form.
Step 3.1:Look at the search page. Notice that you can go to a tutorial on the performance study or these step-by-step instructions. The legend shows icons for information about each data file. Stacked data comes from scintillators that are stacked; Unstacked data is for scintillators set up as an array. The Comments icon, the bubble, lets you see or add comments on a data file.
Step 3.2:Search first for a data file from Seaman High School with date starting with 2/05/2009.
a) Select School in the pulldown list
b) Enter Seaman High School in the search field
b) Click on the little arrow next to "Advanced Search".
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "2/05/2009" in the Start Date field.
b) Click Search Data
Step 3.4:a) Click on the little arrow next to "Seaman High School" to see the data files associated with Seaman High School for this data range.
Step 3.5:a) Click on the little arrow next to "February 2009" to see the data files for that month.
Step 3.6: All the data files uploaded for February are displayed. You can add comments to the file by clicking on the bubble and look at the data. Notice that all of these data files come from a stacked configuration.
Step 4: a) Select the radio button next to "Thu 05"
b) Click Run performance study
Step 5:Look at the number of events in each channel. Notice that they are all about the same numbers. This shows that the detectors are well-plateaued.
a) Change Bin width (ns) to "2.25"
b) Click Analyze.
Note: If you don't understand what the bin width is, you can always click on the Question Mark next to it. An explanation should pop-up.
Step 5.1:If your analysis takes very little time to run, you may get your plot right away. More often, you get a progress bar displaying how the analysis is doing. You can click on Queue study to allow the analysis to go on in the background.
Step 5.2:You can now click on Analyses on the navigation bar or analysis list in the text to see the progress of you job.
Step 5.3:When your job is done, you will see an entry on the Analysis List like this. You can click on the link and see the plot.
Step 6:Here's the plot of the performance (a histogram showing the frequency of the signal widths). Let's see how this plot looks if we make it smaller.
Click Change.
Step 7:Click on the small arrow to view the Plot Controls.
Step 8:a) Select Small for the size of the plot.
b) Click Analyze.
Step 9:Enter a name for the plot similar, but not the same as the one shown and click Save Plot. If you do this "step-by-step" more than once, choose another name for the plot.
You will have access to this plot for your poster.
Step 10:Note you have successfully saved your plot; Click Close
You can do more plots for other channels or go back to choosing a new data set by clicking Performance. Anytime you can click View Plots to see the plots you have made. If you are curious about what files were made during your analysis, uou can click on Analysis List.
