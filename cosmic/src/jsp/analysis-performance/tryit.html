Step-by-Step Instructions: Performance Study

Data Button - Performance Button - Searching Data Files - Selecting Data - Analyzing
The Plot - View Plot Controls - Set Controls - Saving a Plot - Finishing

Keep two windows open, one with these instructions and the other with the Data web pages. Go back and forth between them. Use the scroll bar to move the instructions. The Performance Study allows you to study the quality of the data. Look for the orange highlights for guidance.

Our goal here is to recreate the first plot used as Figure 1 in the poster titled "Can we detect electrons"? You can access it from posters. The data for this plot is from Fermilab and dated 9/16/04

Step 1:Click on the Data button in the main menu.
Step 2:Click on the Performance button in the submenu. You will see a list of many data files.
Step 3.1:Look at the search page. Notice that you can go to a tutorial on the performance study or these step-by-step instructions. The legend shows icons for information about each data file. Stacked data comes from scintillators that are stacked; Unstacked data is for scintillators setup as an array. You can click on the Comments icon to see or add comments on a data file.
Step 3.2:Search first for a data file from Fermilab with date starting with 9/16/2004.
a) Select School in the pulldown list
b) Enter Fermilab in the search field
b) Click on the little arrow next to "Advanced Search".
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "9/16/2004" in the first date field.
b) Click Search Data
Step 3.4:a) Click on the little arrow next to "Fermilab" to see the data files associated with Fermilab.
Step 3.5:a) Click on the little arrow next to "September 2004" to see the data files for that month.
Step 3.6: Hover over "Thu 16" to see additional information about the data on that date. Notice that channels 1 thru 3 have good data.
Step 4: a) Select the radio button next to "Thu 16"
b) Click Run performance study
Step 5:Look at the number of events in each channel.
a) Change the number of Bins to "40"
b) Click Analyze.
Step 6:Here's the graph of the performance (a histogram showing the frequency of the pulse widths). Let's see how this plot looks if we make it smaller.
Click Change.
Step 7:Click on the small arrow to view the Plot Controls.
Step 8:a) Select Small for the size of the plot.
b) Click Analyze.
Step 9:Enter a name for the plot similar, but not the same as the one shown and click Save Plot. If you do this "step-by-step" more than once, choose another name for the plot.
You will have access to this plot for your poster.
Step 10:Note you have successfully saved your plot; Click Close
You can do more plots for other channels or go back to choosing a new data set by clicking Performance. Anytime you can click View Plots to see the plots you have made.
