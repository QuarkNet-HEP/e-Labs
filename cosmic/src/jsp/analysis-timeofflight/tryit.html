Step-by-Step Instructions: Time of Flight Study

Data Button - Shower Button - Searching Data Files - Selecting Data - Analyzing
Shower Candidates - View Plot Controls - Change Caption - The Plot - Save Plot - Done

Keep two windows open, one with these instructions and the other with the ToF study web pages.
Go back and forth between them.
Use the scroll bar to move the instructions.
Look for the aqua highlights for guidance.

We will create a plot with data from the QuarkNet Traveling Array dated 10/29/2006 to 10/30/2006. QuarkNet Traveling Array has been entered into the e-Lab as a school even though it does not seem like your typical school

Step 1:Click on the Data button in the main menu.
Step 2:Click on the T of F button in the submenu.
Step 3.1:Look at the search page. On the right, there are links to a tutorial, step-by-step instructions, and FAQs. The legend shows icons for information about each data file.
Step 3.2:We want to find all data for DAQ 6203 from Nov. 1, 2015 through Jan. 31, 2016.
a) Select Detector ID in the pulldown list.
b) Enter 6203 in the search field.
c) Click on the little arrow next to "Advanced Search."
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "11/01/2015" in the first date field and "01/30/2016" in the second date field. You can also use the calendar.
b) Select Blessed: All.
c) Click Search Data
Step 3.4:a) Click on the little arrow next to "Fermilab Test Array" to see the data files.
Step 3.5:a) Click on the little arrow next to "November 2015" to see the data files for that month.
Step 3.6:One can view the data by clicking on the icon with the magnifying glass and additional information by clicking on the comments bubble.
Step 4:a) Click the button next to the first Monday, 11/23/2015, data set.
Step 4.1: a) Then click Run ToF study on the right.
Step 4.2:Look at the display of the selected data file. You can view the raw data and check the statistics. Also, make sure the geometry file is there.
Step 5:To understand the input parameters, you can click on the question mark next to each. The first time through, keep all the defaults,
a) Center of graph view: Fermilab Test Array, Batavia, IL. (6203)
b) Event gate (ns): 500.
c) Channel Coincidence: 2.
d) Define Soft Triggers? Require Channels: None.
e) Define Soft Triggers? Veto Channels: None.
f) Click Analyze. We will run the analysis on our local computer. This will take a while, so don't be alarmed if you do not get a plot right away.
Step 5.1:If your analysis takes very little time to run, you may get your plot right away. More often, you get a progress bar displaying how the analysis is doing. You can click on Queue study to allow the analysis to go on in the background. You can also click on Analysis output to see the inner workings. This might help you if you encounter an error in your analysis.
Step 5.2:You can now click on Analyses on the navigation bar to see the progress of you job.
Step 5.3:When your job is done, you will see an entry on the Analysis List like this. You can click on the link and see the plot.
Step 6:On the Time Difference ch4-ch1 plot, click Advanced Controls.
Step 7:a) To change the caption on the plot, you will need to use the plot controls.
Click on the small arrow to view the Plot Controls.
Step 8:a) Change the z-max to 16 and edit the list of data files from the figure caption and replace it with something simpler such as
"Data: QuarkNet Traveling Array
Oct 29-30, 2006
Detectors: 5078, 5080
Keep the other settings
b) Consider executing the analysis on the I2U2 Cluster.
c) ClickAnalyze.
New caption
Step 9:a) Click on the small plot to open a window with this plot.
b) Scroll down in the plot window past the date so you can save the plot permanently.
Step 10:Enter a name for the plot similar, but not the same as the one shown and click Save Plot. If you do this tutorial more than once, choose another name for the plot.
You will have access to this plot for your poster.
Step 11:Note you have successfully saved your plot and click Close.
Also close the window with the plot. You can do more plots for other shower candidates or go back to choosing a new data set by clicking Shower in the menu. Anytime you can click View Plots to see the plots you have made.
