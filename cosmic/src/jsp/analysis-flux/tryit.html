Step-by-Step Instructions: Flux Study

Data Button - Flux Button - Searching Data Files - Selecting Data - Analyzing
The Plot - View Plot Controls - Set Controls - Saving a Plot - Finishing

Keep two windows open, one with these instructions and the other with the flux study web pages.
Go back and forth between them.
Use the scroll bar to move the instructions.
Look for the orange highlights for guidance.

We will create a plot with data from detector 5078 at the Fermilab Test Array dated 5/07/07 through 5/11/07. Fermilab Test Array has been entered into the e-Lab as a school even though it does not seem like your typical school name.

Step 1:Click on the Data button in the main menu.
Step 2:Click on the Flux button in the submenu.
Step 3.1:Look at the search page. On the right is a button to run the study and links to a tutorial, step-by-step instructions, FAQs and related milestones. The legend shows icons for information about each data file.
Step 3.2:Search first for data files from Fermilab Test Array with date starting with 5/1/2007.
a) Select School in the pulldown list.
b) Enter Fermilab Test Array in the search field.
b) Click on the little arrow next to "Advanced Search."
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "5/1/2007" in the first date field and "5/30/07" in the second date field. You can also use the calendar to select these dates.
b) Select Yes in the pull-down list for Stacked.
c) Click Search Data.
Step 3.4:a) Click on the little arrow next to "Fermilab Test Array" to see the data files.
Step 3.5:a) Click on the little arrow next to "May 2007" to see the data files for that month.
Step 3.6: All the data files uploaded for May 2007 are displayed. Notice there are data from two detectors. See the data by clicking on the date and additional information by clicking on the comments bubble.
Step 4:We are going to select data from detector with id 5078. a) To select data for analysis, select the checkboxes next to "Mon 07," "Tues 08," "Wed 09." "Thu 10," and "Fri 11."
b) Click Run flux study
Step 5: We will analyze events in channel 1. See the Analysis Controls below.
 We will change the bin width to 7200.
  We will run the analysis on our local computer. See Execution Mode below.
Click Analyze below the Execution Controls.
Step 5.1:If your analysis takes very little time to run, you may get your plot right away. More often, you get a progress bar displaying how the analysis is doing. You can click on Queue study to allow the analysis to go on in the background.
Step 5.2:You can click on Analyses on the navigation bar or analysis list in the text to see the progress of your job.
Step 5.3:When your job is done, you will see an entry on the Analysis List like this. You can click on the link and see the plot.
Step 6:Here's the graph of the flux (a histogram showing the number of events/m2/60 seconds from May 7, 2007 through May 11, 2007. The legend for the data files covers some of the points so you should change the maximum y value in the plot.
Click Change your parameters to alter the plot.
Step 7:Click on the small arrow to view the Plot Controls.
Step 8:Try varying some of the plot controls.
a) Enter 650 as Y-max
b) Try executing on the I2U2 Cluster instead of on the local server and see if take more or less time.
c) Click Analyze.
Step 8.5:You will see the progress bar, can queue the study and then go to it from the analysis list. See how much time it took compared the study you ran when you executed it on the local server. As described above, click on the study name when it is completed to see the plot.
Step 9:Here's your new plot. The legend for the data files clears the data points. Now you can save it.
Enter a name for the plot and click Save Plot. You should choose a different name each time you do the tutorial.
You will have access to this plot for your poster.
Step 10:Note you have successfully saved your plot and click Close.
You can do more plots for other channels or go back to choosing a new data set by clicking Flux. Anytime you can click View Plots to see the plots you have made. You can also run any study again from the plot you saved. You may want to see if it runs faster or slower when you execute it on the grid.
