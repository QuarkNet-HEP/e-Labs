Step-by-Step Instructions: Shower Study

Data Button - Shower Button - Searching Data Files - Selecting Data - Analyzing
Shower Candidates - View Plot Controls - Change Caption - The Plot - Save Plot - Done

Keep two windows open, one with these instructions and the other with the actual web pages. Go back and forth between them. Use the scroll bar to move the instructions. Look for the orange highlights for guidance.
Step 1:Click on the Data button in the main menu.
Step 2:Click on the Shower button in the submenu.
Step 3.1:Look at the search page. Notice that you can go to a tutorial on the shower study or these step-by-step instructions. The legend shows icons for information about each data file. Stacked data comes from scintillators that are stacked; Unstacked data is for scintillators setup as an array. You can click on the Comments icon to see or add comments on a data file.
Step 3.2:We want to find all data for Washington state for dates Oct. 19-21, 2004. We decided on these dates because there was overlap in the data during this period.
a) Select State in the pulldown list
b) Enter WA in the search field.
b) Click on the little arrow next to "Advanced Search".
Step 3.3:a) Select Start Date in the pulldown list for dates and enter "10/19/2004" in the first date field and 10/21/2004 in the second date field.
b) Click Search Data
Step 3.4:a) Click on the little arrow next to "DeVry" to see the data files associated with DeVry.
Step 3.5:a) Click on the little arrow next to "October 2004" to see the data files for that month.
Step 3.6: Hover over "Tue 19" to see additional information about the data on that date. Notice all channels have good data.
Step 4:a) Click the checkbox labeled "Select all 3 files"
.
Step 4.1:a) Click the triangles on all schools and months and click the select all checkbox so you have selected every data file.
b) Then click Run shower study on the right.
Step 4.2:Look at the display of all the data files you have selected. You can view the raw data, check the statistics, and compare them by using the links on the right.
Step 5:a) Set the 0,0,0 point, the center of the detectors to the detector id 141
b) the event gate to 10000000 (7 zeroes makes .01 seconds, a wide gate)
c) the event coincidence level to 10 (number of detector hits required to define an event)
d) the Detector Coincidence to 3
e) Click Analyze. This will take awhile so don't be alarmed if you do not get a plot right away.
Step 6:Here are a list of the possible shower candidates. The plot of the first one, highlighted in blue, is on the right. The data table shows all the events associated with this candidate.
a) Click on another candidate and wait for the plot to show.
b) Click on the first candidate again
Notice that the caption is really too large.
c) Click on Change at the bottom to go back and fix caption in the plot parameters.
Step 7:a) To change the caption on the plot, you will need to use the plot controls.
Click on the small arrow to view the Plot Controls.
Step 8:a) Cut out the list of data files and data files from the caption and replace it with something simpler such as "Washington State, Oct. 19-21, 2004 (110,141,150); Keep the other settings.
b) Click Analyze.
New caption
Step 9:a) Click on the small plot to open a window with this plot.
b) Scroll down in the plot window past the date so you can save the plot permanently.
Step 10:Enter a name for the plot similar, but not the same as the one shown and click Save Plot. If you do this tutorial more than once, choose another name for the plot.
You will have access to this plot for your poster.
Step 11:Note you have successfully saved your plot and click Close.
Also close the window with the plot. You can do more plots for other shower candidates or go back to choosing a new data set by clicking Shower in the menu. Anytime you can click View Plots to see the plots you have made.
