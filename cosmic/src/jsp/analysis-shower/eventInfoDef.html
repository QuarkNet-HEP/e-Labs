Event Information Definitions

Information about events that satisfy your search criteria are listed in three columns with one line per event.

  1. If you click on the headings, events are reordered.
    • Event Date: The date and UTC (Greenwich Mean) time in hr:min:sec.
    • Hit Coincidence: Total number of counter hits.
    • Detector Coincidence: Number of detectors hit.
      • DAQi plus number of counters in DAQi that fire (can be less than number of hits if a counter fires multiple times).
      • DAQj plus number of counters in DAQj that fire.
  2. If you click on the advanced topic "View Multiplicity" a new column appears summing the total number of hit counters.
  3. The total number of events is listed in the top heading as "Number shower study candidates."
  4. "Event List Reference" links to shower tutorial.
  5. An event display is provided for any event you select. The detailed information for that display appears in a table below providing position, time of counter hits, detector and channel.
